Bootstrap Basics Challenge

HTML/CSS -- Level 2

Requirements

  • HTML:
    • Create 3 section tags
    • Use the bootstrap structure to created nested divs within each section
      • i.e. container, row, col-md-*
  • CSS:
    • Center all the text
    • Set the height of each column to anything between 100px to 500px
    • Add a 2px border to each column with any color and style of your choosing
    • For each section use alternating background colors
    • Also, give padding to the top and bottom of each section
    • Use alternating background colors for each row
