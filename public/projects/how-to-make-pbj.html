How to Make PBJ Challenge

HTML/CSS -- Level 1

Requirements

  • HTML:
    • Use h1 and h2 tags for the page title
    • Use either an ol or ul tag with nested li tags
    • Use a tags to turn each img tag into an external hyperlink
      • link a tag to either a Wiki Page or Youtube Video about the image
  • CSS:
    • Use a pattern for the background in the body
    • Center all the text on the page
    • Set li tags to display inline
    • Set all the img tags to the same height
    • Add borders and hover properties to img tags
